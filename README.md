# REST API сервер на чистом Node.js для управления библиотекой книг. Реализованы CRUD-операции, фильтрация по категории и хранение данных в локальном JSON-файле
____
## :memo: Функционал:
- Получение списка книг
- Фильтрация по категории
- Добавление новой книги
- Обновление статуса прочтения
- Удаление книги
- Валидация входных данных
- Сохранение данных в JSON
## :file_folder: Структура проекта

```
├── index.js        # Основной сервер
├── books.json      # Файл хранения данных
└── index.html      # HTML-страница (главная)
```
